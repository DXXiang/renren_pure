## 如何合作开发？

#### 1. 成为Collaborator

* 在项目的issue中给我留言，我会向你发送邀请成为合作开发者
* 在你的绑定邮箱中确认接受我的邀请

#### 2. 或者仅仅fork这个项目为你的repository

* 在你开发完成后向我发出**pull request**

#### 3. 建议

* 无论什么方式，**建议在本地my_dev分支开发，然后push到dev分支上**，最后我们再决定如何merge到master分支
